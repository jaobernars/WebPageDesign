Introduction to Python

Um guia completo e progressivo para aprender Python do zero, cobrindo desde a instalação e sintaxe básica até estruturas de dados, funções, orientação a objetos e bibliotecas científicas essenciais para a física computacional.

Resumo

Este material introdutório foi elaborado para fornecer uma base sólida na linguagem de programação Python, voltada especialmente para estudantes de ciências exatas e engenharias. O conteúdo percorre os fundamentos da linguagem — variáveis, tipos de dados, controle de fluxo e funções — avançando progressivamente para estruturas de dados, manipulação de arquivos, orientação a objetos e, por fim, uma introdução às bibliotecas NumPy, Matplotlib e SciPy, ferramentas indispensáveis para a computação científica.

Fundamentos da Linguagem

Sintaxe, variáveis, tipos primitivos, operadores, strings, entrada/saída e boas práticas de código Python.

Estruturas de Dados

Listas, tuplas, dicionários, conjuntos, list comprehensions e manipulação eficiente de coleções.

Computação Científica

Introdução ao NumPy para álgebra linear, Matplotlib para visualização e SciPy para resolução numérica de EDOs e integrais.
